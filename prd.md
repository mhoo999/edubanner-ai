## 1. 제품 개요

**제품명**: EduBanner AI

**목적**: 배너 제작 시 AI가 테마와 상세 레이아웃을 텍스트로 제공

**타겟**: 메가존 교육사업부 디자인팀

**기술**: Next.js + Claude API

---

## 2. 핵심 기능

### 2.1 사용자 플로우

```
Step 1: 테마 여부 확인
  ├─ [예] → 테마 선택 → Step 3
  └─ [아니오] → Step 2

Step 2: 테마 추천 (3개)
  - 용도/주제/분위기 입력
  - AI가 3개 추천 (텍스트 설명만)
  - 하나 선택

Step 3: 상세 정보 입력
  - 선택된 테마 표시
  - 배너 크기 (커스텀 입력)
  - 메인/서브 카피
  - 이미지 업로드 (선택)

Step 4: 레이아웃 생성
  - 텍스트 결과 출력
  - 복사 버튼

```

### 2.2 테마 추천 로직

**입력 → AI 분석 → 3개 추천**

예시:

```
입력:
- 용도: 강좌 홍보
- 주제: AWS 클라우드
- 분위기: 전문적

출력:
1. 모던 테크 (매칭 85%)
   - 컨셉: 기술 혁신, 전문성
   - 색상: 다크블루 + 시안

2. 신뢰감 블루 (매칭 78%)
   - 컨셉: 안정감, 교육 전문성
   - 색상: 진파랑 + 화이트

3. 다크 엘레강스 (매칭 72%)
   - 컨셉: 고급스러움, 프리미엄
   - 색상: 블랙 + 골드

```

### 2.3 레이아웃 출력 형식

```markdown
## 선택된 테마: 모던 테크

### 배경
- 타입: 대각선 그라데이션
- 색상: #1a2332 → #2d3e50
- 각도: 135도

### 요소 배치

**메인 카피** "AWS 실습 과정"
- 위치: 중앙 (X: 50%, Y: 40%)
- 폰트: Pretendard Bold 72px
- 색상: #ffffff
- 효과: 드롭섀도우 (0, 4px, 20px, rgba(0,0,0,0.3))

**서브 카피** "12월 개강"
- 위치: 중앙 (X: 50%, Y: 55%)
- 폰트: Pretendard Medium 36px
- 색상: #4fc3f7

**업로드 이미지 (로고)**
- 위치: 좌상단 (X: 60px, Y: 50px)
- 크기: 150x50px

[전체 텍스트 복사]

```

## 3. 기술 스택

### 3.1 프론트엔드

- **Framework**: Next.js 14 (App Router)
- **UI**: Tailwind CSS
- **배포**: Vercel

### 3.2 백엔드

- **API**: Next.js API Routes (`/api/recommend`, `/api/generate`)
- **AI**: Claude 3.5 Sonnet API
- **환경변수**: `.env.local` (로컬), Vercel 환경변수 (배포)

### 3.3 주요 라이브러리

```json
{
  "@anthropic-ai/sdk": "^0.30.0",
  "next": "^14.0.0",
  "react": "^18.0.0",
  "tailwindcss": "^3.4.0"
}

```

## 4. API 설계

### 4.1 테마 추천 API

**Endpoint**: `POST /api/recommend`

**Request**

```json
{
  "purpose": "강좌 홍보",
  "keywords": "AWS 클라우드",
  "mood": "전문적"
}

```

**Response**

```json
{
  "themes": [
    {
      "name": "모던 테크",
      "score": 85,
      "concept": "기술 혁신, 전문성",
      "colors": "다크블루 + 시안"
    },
    {
      "name": "신뢰감 블루",
      "score": 78,
      "concept": "안정감, 교육 전문성",
      "colors": "진파랑 + 화이트"
    },
    {
      "name": "다크 엘레강스",
      "score": 72,
      "concept": "고급스러움, 프리미엄",
      "colors": "블랙 + 골드"
    }
  ]
}

```

### 4.2 레이아웃 생성 API

**Endpoint**: `POST /api/generate`

**Request**

```json
{
  "theme": "모던 테크",
  "size": {
    "width": 1200,
    "height": 628
  },
  "mainCopy": "AWS 실습 과정",
  "subCopies": ["12월 개강", "선착순 30명"],
  "hasImage": true,
  "imageType": "로고"
}

```

**Response**

```json
{
  "layout": "## 선택된 테마: 모던 테크\n\n### 배경\n- 타입: 대각선 그라데이션\n..."
}

```
